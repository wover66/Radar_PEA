# Radar PEA

Application Streamlit d'analyse d'actions européennes éligibles au PEA, avec critères GARP (Growth At a Reasonable Price).

## Fonctionnalités

- Analyse des actions PEA selon des critères fondamentaux (PER, croissance, valorisation)
- Onglet crypto
- Gestion de portefeuille
- Mise en cache des données

## Lancement

```bash
streamlit run main_app.py
```
